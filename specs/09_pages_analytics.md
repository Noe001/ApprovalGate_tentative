# アナリティクス ページ仕様

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/t/:tenant_id/analytics` |
| アクセス権限 | Admin+ / Viewer |
| 目的 | AIエージェントの利用状況・承認傾向・応答時間を可視化し、ルール最適化の判断材料を提供する |

---

## レイアウト構成

```
┌────────────────────────────────────────────────────────────┐
│ アナリティクス              [期間: 過去30日間 ▼]  [CSVエクスポート] │
├────────────────────────────────────────────────────────────┤
│ KPIサマリー（4枚）                                            │
├────────────────────────────────────────────────────────────┤
│ リクエスト量の推移（折れ線グラフ）                               │
├─────────────────────┬──────────────────────────────────────┤
│ ステータス内訳（ドーナツ）│ プロジェクト別件数（横棒グラフ）          │
├─────────────────────┴──────────────────────────────────────┤
│ 承認者別パフォーマンス（テーブル）                               │
├────────────────────────────────────────────────────────────┤
│ 時間帯別・曜日別ヒートマップ                                     │
└────────────────────────────────────────────────────────────┘
```

---

## 期間選択

| 選択肢 | 内容 |
|--------|------|
| 今日 | 本日0時から現在まで |
| 過去7日間 | 直近7日間 |
| 過去30日間 | 直近30日間（デフォルト） |
| 過去90日間 | 直近3ヶ月 |
| カスタム | 開始日〜終了日を日付ピッカーで指定 |

期間変更時は全グラフ・テーブルが即座に更新される。

---

## KPIサマリー（4枚）

### 総リクエスト数

- 選択期間内の全承認リクエスト件数
- 前期間比（例: 「前月比 +23%」）

### 承認率

- （承認済み件数 / 全完了件数）× 100%
- 自動承認・手動承認の両方を含む
- サブ: 「うち自動承認 XX%」

### 平均応答時間

- 人間が承認・却下するまでの平均時間（自動承認は除外）
- 前期間比

### タイムアウト率

- タイムアウトした件数 / 全件数 × 100%
- 高い場合は警告色（10%以上でオレンジ、20%以上で赤）

---

## リクエスト量の推移（折れ線グラフ）

- X軸: 日付（期間によって日・週・月単位で自動調整）
- Y軸: リクエスト件数
- 複数の系列:
  - 全リクエスト（灰色の細線）
  - 手動承認（緑）
  - 自動承認（青）
  - 却下（赤）
  - タイムアウト（グレー）
- 各系列はクリックで表示・非表示を切り替え可能
- ホバーでその日の各値をツールチップ表示

---

## ステータス内訳（ドーナツグラフ）

選択期間内のリクエストのステータス割合を表示。

- セグメント: 承認（緑）/ 自動承認（青）/ 却下（赤）/ タイムアウト（グレー）/ 承認待ち（黄）
- 中央に総件数を表示
- 凡例をクリックでセグメント非表示

---

## プロジェクト別件数（横棒グラフ）

テナント内の各プロジェクトの件数を比較。

- Y軸: プロジェクト名（件数の多い順にソート）
- X軸: 件数
- 各棒はステータスで色分け（積み上げ棒グラフ）
- プロジェクト名クリックでそのプロジェクト詳細に遷移

---

## 承認者別パフォーマンス（テーブル）

| 列名 | 内容 |
|------|------|
| 承認者名 | メンバー名 |
| 処理件数 | 選択期間内に処理した件数（承認+却下） |
| 承認率 | 処理件数のうち承認の割合 |
| 平均応答時間 | 通知受信から承認・却下までの平均時間 |
| タイムアウト件数 | 応答しなかった件数 |

- 処理件数でデフォルトソート
- 列ヘッダークリックでソート順を切り替え

---

## 時間帯別・曜日別ヒートマップ

リクエストが多い時間帯・曜日を可視化。ルールのタイムアウト設定やエスカレーション設定の参考に使う。

```
     月   火   水   木   金   土   日
0時  ·    ·    ·    ·    ·    ·    ·
6時  ·    ·    ·    ·    ·    ·    ·
9時  ■    ■    ■    ■    ■    ·    ·    ← 濃い = リクエスト多い
12時 ■■   ■■   ■■   ■■   ■■   ·    ·
18時 ■    ■    ■    ■    ■    ·    ·
21時 ·    ·    ·    ·    ·    ·    ·
```

- ホバーでその時間帯の件数・承認率を表示

## CSVエクスポート

「CSVエクスポート」ボタンでダウンロード。

| エクスポートに含まれる項目 |
|------------------------|
| リクエストID |
| プロジェクト名 |
| アクション名 |
| ステータス |
| 担当承認者 |
| 受信日時 |
| 完了日時 |
| 応答時間（秒） |
| 却下理由（あれば） |
| 自動承認の場合: 適用ルール名 |
