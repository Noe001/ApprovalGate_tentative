# ダッシュボード（ホーム）ページ仕様

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | `/t/:tenant_id/dashboard` |
| アクセス権限 | 全ロール（表示内容はロールで変化） |
| 目的 | テナント全体の状況を一目で把握し、緊急対応が必要な承認リクエストへ素早くアクセスする |

---

## Owner / Admin 向けダッシュボード

### レイアウト構成

```
┌─────────────────────────────────────────────────────┐
│ ダッシュボード                          [期間: 今日 ▼] │
├──────────┬──────────┬──────────┬────────────────────┤
│ 承認待ち  │ 今日の処理 │ 自動承認率 │ 平均応答時間        │
│  3 件    │  47 件   │  93.2%  │  4分12秒           │
├──────────┴──────────┴──────────┴────────────────────┤
│ 承認待ちリクエスト                          [すべて見る] │
├─────────────────────────────────────────────────────┤
│ [!] メール送信 / marketing-agent / 2分前    [承認][却下] │
│ [!] API呼び出し / billing-agent / 5分前    [承認][却下] │
│ [!] ファイル削除 / cleanup-agent / 8分前   [承認][却下] │
├─────────────────────────────────────────────────────┤
│ 今日のアクティビティ                                    │
├─────────────────────────────────────────────────────┤
│ [グラフ: 時間帯別処理件数]                              │
└─────────────────────────────────────────────────────┘
```

---

### KPIカード（4枚）

#### 承認待ち件数

- 数値: 現在の承認待ちリクエスト総数
- クリック: 承認リクエスト一覧ページ（「承認待ち」フィルター済み）に遷移
- 強調表示: 件数が多い場合（プランで設定した閾値超）は黄色の背景

#### 今日の処理件数

- 数値: 当日0時から現時点までに完了したリクエスト数（承認・却下・自動承認・タイムアウト合計）
- サブテキスト: 前日比を表示（「昨日比 +12件」）

#### 自動承認率

- 数値: 過去24時間の全リクエストのうち自動承認された割合（%）
- 説明ツールチップ: 「ルールにより人間の確認なく承認されたリクエストの割合です」

#### 平均応答時間

- 数値: 過去24時間の人間が承認・却下するまでの平均時間
- 自動承認は含まない
- サブテキスト: 「最長: 32分15秒」

#### KPIカードの期間選択

- ドロップダウンで切り替え: 今日 / 過去7日間 / 過去30日間
- 切り替えると全カードの値が更新される

---

### 承認待ちリクエスト一覧（最大5件）

最も古い承認待ちリクエストを上から表示（先着優先）。

#### リクエストカードの内容

| 要素 | 内容 |
|------|------|
| ステータスアイコン | 黄色の「!」マーク |
| アクション名 | 例: 「メール送信」「API呼び出し」「ファイル削除」 |
| プロジェクト名 | 例: 「marketing-agent」 |
| 経過時間 | 例: 「2分前」（リアルタイム更新） |
| タイムアウトまでの時間 | 例: 「残り22分」（設定されている場合） |
| 承認ボタン | 緑色・クリックで即時承認（確認なし） |
| 却下ボタン | 赤色・クリックで却下理由入力モーダルを表示 |

#### 「すべて見る」リンク

承認リクエスト一覧ページ（フィルター: 承認待ち）に遷移。

#### 空の状態

```
✅ 承認待ちのリクエストはありません
   現在すべてのリクエストが処理されています
```

---

### 今日のアクティビティグラフ

**グラフ仕様**

- 種別: 棒グラフ（積み上げ）
- X軸: 0時〜24時（1時間単位）
- Y軸: 件数
- 色分け: 承認済み（緑）/ 却下（赤）/ 自動承認（青）/ タイムアウト（グレー）
- ホバー: その時間帯の各ステータス件数をツールチップ表示

---

### 最近のアクティビティフィード（右サイドまたは下部）

直近20件のアクティビティを時系列で表示。

```
● 田中さんが「メール送信」を承認しました       2分前
● 「API呼び出し」がタイムアウトしました        5分前
● 鈴木さんが「ファイル削除」を却下しました     12分前
● ルール「高額支払い自動承認」が適用されました  15分前
```

## Approver 向けダッシュボード

管理系情報は非表示。自分の担当リクエストに集中したレイアウト。

```
┌─────────────────────────────────────────────────────┐
│ ダッシュボード                                         │
├──────────────┬──────────────┬────────────────────────┤
│ 自分の担当待ち │ 今日処理した件数│                       │
│    2 件      │    8 件      │                       │
├──────────────┴──────────────┤                       │
│ 自分の承認待ちリクエスト                                 │
├─────────────────────────────────────────────────────┤
│ [!] メール送信 / 3分前              [承認] [却下]        │
│ [!] ファイル更新 / 11分前           [承認] [却下]        │
└─────────────────────────────────────────────────────┘
```

---

## Viewer 向けダッシュボード

承認ボタン・KPIの詳細数値は表示するが、操作不可。統計グラフのみ表示。

---

## ダッシュボードの自動更新

- 承認待ち件数・KPIカードは **30秒ごと**に自動更新
- 承認操作後はリアルタイムでリスト・件数が更新される
- バックグラウンドタブでも更新は継続する
- 更新中はカードの右上にスピナーを微小表示
